Apache httpd : Install httpd
2014/07/18
 
Install httpd to configure Web Server. HTTP uses 80/TCP.
[1] Install httpd.
[root@www ~]#
yum -y install httpd
# remove welcome page

[root@www ~]#
rm -f /etc/httpd/conf.d/welcome.conf
[2] Configure httpd. Replace server name to your own environment.
